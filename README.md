## О чем проект

В рамках проекта разрабатывается прототип системы автоматизации email-маркетинга, адаптированный для учебных и демонстрационных целей. Ключевая идея — смоделировать полный жизненный цикл рассылки:
-	Подготовка контента — создание и редактирование HTML-шаблонов писем с функцией предпросмотра.
-	Настройка кампании — выбор шаблона, целевого сегмента клиентов (из синтетического справочника) и планирование запуска.
-	Имитация отправки — для каждого клиента из сегмента с подтверждённым согласием (consent_flag=true) вызывается REST-мок-метод POST /mail/send, возвращающий статус sent или failed.
-	Логирование и контроль — все попытки фиксируются в журнале доставки с временными метками и кодами ошибок.
-	Аналитика — формирование отчёта с KPI (доля успешных отправок, разрезы по сегментам и кампаниям) и экспорт данных в CSV.
